Adding a < caption > to a Table

  • The < caption > element appears directly after the opening < table > tag; it should come before the first row:
  • < table > < caption > Opening hours for the Example Cafe< /caption > < tr >
  • By using a < caption > element, rather than just describing the purpose of the table in a previous or subsequent paragraph, you are directly associating the content of the table with this description
  • EXAMPLE:
    	< table border ="1"width="30%" style="border-collapse:collapse;border-color:red" >
		< caption>THIS IS EXAMPLE FOR CAPTION< /caption>
		< tr>
			< th>1 < /th>
			< th>2 < /th>
		< /tr>
		< tr>
			< td>3 < /td>
			< td>4 < /td>
		< /tr>
		< tr>
			< td>5 < /td>
			< td>6 < /td>
			< /tr>
	     < br>
	 < /table>

Grouping Sections of a Table

  • Use the rowspan and colspan attributes to make cells stretch over more than one row or column
  • Split a table into three sections: a head, body, and foot
  • Group columns using the < colgroup > element
  • Share attributes between unrelated columns using the < col > elementv
  • EXAMPLE FOR COLSPAN AND ROWSPAN: 
    < table border ="1"width="20%" style="border-collapse:collapse;border-color:blue" >
		< tr>
			< th rowspan="2">1< /th>
			< th colspan="3">2< /th>
		< /tr>
	 	< tr>
			< td>3< /td>
			< td>4< /td>
		< /tr>
		< tr>
			< td>5< /td>
