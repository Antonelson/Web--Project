TABLE

Basic Table Elements and Attributes

  • Tables display information in rows and columns; they are commonly used to display all manner of data that fits in a grid such as
    1. train schedules
    2. television listings
    3. financial reportssports results

  • Each cell must be represented by either a < td > or a < th > element in order for the table to display correctly, even if there is no data in that cell

  • All tables will follow this basic structure, although there are additional elements and attributes that allow you to control the presentation of tables. If a row or column should contain a heading, a < th > element should be used in place of the < td > element for the cells that contain a heading. By default, most browsers render the content of a < th > element in bold text.

  • Basic Table Elements and Attributes
    • The < table > Element Creates a Table
    • The align Attribute (Deprecated).it indicates whether the table should be aligned to the left (the default), right , or center of the page.
    • The bgcolor Attribute (Deprecated) .sets the background color for the table
    • The border Attribute (Deprecated).a border will be created around both the table and each individual cell.
    • The cellpadding Attribute (Deprecated) is used to create a gap between the edges of a cell and its contents
    • The cellspacing attribute is used to create a space between the borders of each cell.
    • The < tr > Element Contains Table Rows
    • The align attribute specifies the position of the content of all of the cells in the row.
    • The < td > and < th > Elements Represent Table Cells
    • The colspan attribute is used when a cell should span across more than one column.
    • The rowspan Attribute specifies the number of rows of the table a cell will span across, the value of the attribute being the number of rows the cell stretches across
