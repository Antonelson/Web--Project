Introducing the box model

  • The box model is a very important concept in CSS because it determines how elements are positioned within the browser window. It gets its name because CSS treats every element as if it were in a box .

  • Every box has three properties border, margin , padding.

  • Example: 
    
< !DOCTYPE html>
< html>
< head>
< title>CSS Box Model Example< /title>
< style>
.box {
width: 300px;
height: 300px;
padding: 30px;
border: 3px solid black;
margin-left: 500px;
background-color: gold; 
}
< /style >
< /head>
< body>
< p class="box">This is a box with a gold background.< /p>
< /div>
< /body>
< /html>
