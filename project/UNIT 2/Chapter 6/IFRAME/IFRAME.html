IFRAME

  • The < iframe > element sits in the middle of a normal XHTML page to create an inline frame
  • The only attribute it must carry is the src attribute, whose value is the URL of the page to be included (wherever the < iframe > element is in the document), although it is also good to add the height and width attributes to control its size. Remember that this element does not have to be part of the frameset document type.
  • The src attribute is required on the < iframe > element as it indicates where the browser can find the page to show in the iframe
  • The height and width attributes enable you to specify the height and width of a frame just as you would with an image.
