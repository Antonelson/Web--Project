List

  • HTML lists allow web developers to group a set of related items in lists.

  • An unordered list starts with the < ul > tag. Each list item starts with the < li > tag.

  • you can change bullet points of list by type="".

  • The values are:
    • disc(default)
    • circle
    • square
    • none

  • An ordered list starts with the < ol > tag. Each list item starts with the < li > tag.

  • you can change bullet points of list by type="".

  • The values are:
    • Numbers
    • Letters(uppercase/lowercase)
    • Roman(uppercase/lowecase)

  • Example: < ol>
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ol>
    < ol type="I">
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ol>
    < ol type="A">
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ol>
    < ul>
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ul>
    < ul type="square">
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ul>
    < ul type="circle">
    < li>1st Element< /li>
    < li>2nd Element< /li>
    < /ul>
